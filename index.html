Open Source

Anima is a meritocratic community interested in developing an OS and apps with modern web design in mind. We believe in the power of Open Source and worldwide collaboration, as we want to build, share and improve software together and make it available to everyone.

Apps

Anima ships with a set of modern and flexible apps. There are core apps very tightly integrated into the OS and cross-platform web apps that can be used on all operating systems and form factors.
